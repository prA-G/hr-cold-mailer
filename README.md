A reusable web app to send personalized cold emails to HRs safely in batches.

## Features
- Excel upload (name, company, email)
- Resume attachment
- Batch-based sending
- Anti-spam delays
- Gmail App Password support
- No data stored

## How to Use
1. Upload Excel
2. Upload Resume (PDF)
3. Enter email + app password
4. Write cold email using {name} & {company}
5. Choose batch size
6. Click Send

## Tech Stack
- Python
- Streamlit
- Pandas
